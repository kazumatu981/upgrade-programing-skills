# `EventHandler` クラス

## 概要

イベントを管理する基底クラス。
本プロジェクトで活用する部品はこれを継承して作成すること。

## constructor

`_events` を空の連想配列として準備する

## プロパティ

なし

## メソッド

### `on(eventName, eventHandler)`

イベントを登録する。
ローカル変数に `eventHandler` を登録しておくことで、`fire()`が呼び出されるとそのメソッドが実行されるように予約しておく。

同じイベント名が指定されても、イベントは配列で格納されるため、上書きされることはない。

#### 引数

- `eventName`: `string` - 追加するイベント情報
- `eventHandler`: `function` - イベント発生時に呼び出す関数

#### 戻り値

なし

#### 処理概要

- クラスローカル変数 `_events` 内を `eventName` をキーにして検索する
- **見つからなかった場合:** 新しい配列を作成して、その配列を `_events` に格納する
- **見つかった場合:** 配列に eventHandlerを追加する

#### 例外

- `eventName` の型が `string` ではなかった場合
- `eventHandler` の型が `function` ではなかった場合
