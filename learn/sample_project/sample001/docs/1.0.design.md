# 設計書

## 画面レイアウトおよび部品構成

![画面レイアウト](./img/1.0.layout.drawio.svg)

このアプリケーションには数え上げた数字を表現する **テキスト領域** と、その数字を操作する**ボタン群**から成り立っている。

それらは、図のように、**M1.カウンタ** と **M2.コントローラ** の二つの部品から構成する設計とする。

## 部品関連図

### 部品階層

`HTML` の DOM 構成は下記のような階層構造とする。

```text
<app-root>
    + M1.カウンタ             <div id='counter' />
    + M2.コントローラ         <div id='controller' />
        + M2-1.プラスボタン   <div id='increment' class='button' />
        + M2-2.マイナスボタン <div id='decrement' class='button' />
```

また、「M2-1.プラスボタン」と「M2-2.マイナスボタン」のスタイルを統一化できるように、
`button`という名称のクラスを定義する

### 部品動的関連

![コールグラフ](./img/1.0.call-graph.drawio.svg)

「M1.カウンタ」を実装する部品と、「M2.コントローラ」を実装する部品には
それぞれ以下のイベントハンドラを準備しておき、そのイベントが発生した場合、他方のメソッドを実行するまたは、プロパティを変更する設計とする。
