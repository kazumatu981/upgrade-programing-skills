# 部品設計書: M1.コントローラ部品: `Controller`

---

## `constructor(incrementButtonId, decrementButtonId)`

- `counterId` を検索キーとしてDOM要素を検索し、クラスローカル変数に格納する。
- オブジェクトが見つからない場合は例外を発生させる。

## プロパティ

### `enableIncrement`

プラスボタンの活性/非活性

#### `set enableIncrement(value)`

- プラスボタンの活性状態(ローカル変数 `_enableIncrement`)を設定する。
- `value` が `true` の場合 DOMのクラスに `enableButton` を追加し、 `disableButton` を削除する
- `value` が `false` の場合 DOMのクラスに `disableButton` を追加し、 `enableButton` を削除する

#### `get enableIncrement()`

プラスボタンの活性状態(ローカル変数 `_enableIncrement`)を参照する

### `enableDecrement`

マイナスボタンを活性/非活性

#### `set enableDecrement(value)`

- プラスボタンの活性状態(ローカル変数 `_enableDecrement`)を設定する。
- `value` が `true` の場合 DOMのクラスに `enableButton` を追加し、 `disableButton` を削除する
- `value` が `false` の場合 DOMのクラスに `disableButton` を追加し、 `enableButton` を削除する

#### `get enableDecrement()`

プラスボタンの活性状態(ローカル変数 `_enableDecrement`)を参照する

## メソッド

なし

## イベント

### `click_increment`

プラスボタンをクリックしたとき

#### 引数

- `thisObject`: `Controller` - イベントが発生したオブジェクト

### `click_decrement`

マイナスボタンをクリックしたとき

#### 引数

- `thisObject`: `Controller` - イベントが発生したオブジェクト
