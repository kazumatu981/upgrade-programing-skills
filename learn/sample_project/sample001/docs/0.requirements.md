# 要件定義

## アプリケーション名

99までカウンタ

## アプリケーション概要

- このアプリケーションは、画面を操作することで、数字を数え上げるアプリケーションである。
- この度、XXXXX高校のYYYY部は文化祭のイベントで模擬店をすることとなった。
- 模擬店の来場者数をカウントするため、このアプリケーションを開発することとなった。
- `+` ボタンと `-` ボタンがある
- `+` ボタンをクリックすると、数字が増加し、 `-` ボタンをクリックすると数字が減少する
- `-` ボタンは、誤って `+` を押しすぎた場合の救済措置とする。

## アプリケーション開発に向けた制約

- 模擬店で使用するコンピュータは Windows 11 とし、前提とするアプリケーションの導入を極力避ける。
- Webブラウザ上で動作するアプリケーションとする
- このアプリケーションは、来年度以降も利用するため、引き継ぐことを前提として以下の点に注意して開発すること
    - どのような意図で開発したのかを明確にしておくこと
    - 他者が見て、わかりやすい(意図が伝わりやすい)コードを記述すること
    - どのようなテストを実施して、どのような問題点が発生したかを記録しておくこと
